# Ours

**Tagline:** Stay close. Even when apart.

Web-first relationship app for long-distance couples focused on calm, low-friction connection.

## Core emotional outcome
"I feel chosen."

## v1 Audience
Long-distance couples with busy schedules, kids, work, and/or time-zone differences.

## v1 Platform
- Web app first
- Mobile later

## Core loop
- Lightweight daily connection
- Flexible weekly reset
- Reassurance tools
- No guilt tracking or streak shaming

## MVP status
- Product + architecture docs created
- Supabase schema + seed content drafted
- Ready to scaffold Next.js + Supabase app
