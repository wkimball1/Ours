# Ours Outreach Kit (First 100 Users)

Use these assets to drive the first 100 waitlist + invite activations.

## Files
- `founder-dm-templates.md` — copy/paste outreach scripts
- `partner-invite-messages.md` — in-product invite message options
- `micro-creator-brief.md` — tiny collab brief for creators/newsletters
- `first-100-tracker.csv` — lightweight tracker for outreach execution

## Suggested weekly cadence
1. Send 25 founder DMs
2. Follow up after 48 hours
3. Share 1 creator brief
4. Review conversion by source in analytics_events
